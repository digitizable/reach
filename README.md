# Spectre Desktop

Desktop UI for [Spectre](https://github.com/digitizable/spectre): profiles, connect/disconnect, and path status over the local core.

Linux only for now (GTK 4 + libadwaita). macOS and Windows ports can come later as separate frontends against the same core API.

## Shape

- **Thin client** — does not own paths, keys, or backends; talks to the Spectre core
- **Pages** — Connection, Profiles, Status, Settings
- **Core** — see the [spectre](https://github.com/digitizable/spectre) repository

## Status

Design stage. Shell UI only; the core is not wired yet.
